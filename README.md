FlowStack is a full-stack personal analytics web application that helps you log and visualize your daily behaviors. Instead of scattered notes and apps, FlowStack gives you a unified dashboard to track your habits, focus sessions, decisions, and skills and helps you identify patterns that lead to real improvement.

---

## What Does FlowStack Do?

Most people struggle not because they lack ambition, but because they lack **visibility into their own behavior**. FlowStack solves this by giving you:

- **HabitSync** Track daily habits with streak counting. Shows your weekly consistency as a progress bar. The longer your streak, the stronger the habit.
- **Focus Tracker** Log your deep work sessions with a built-in timer. See how many hours per day and per week you're actually spending on focused work.
- **DecisionLog** Record every significant decision you make and tag it as *good*, *bad*, or *neutral*. Over time, this reveals patterns in your judgment.
- **Skills** Rate your proficiency on any skill from 1вЂ“10. Use this as your personal skill map to see where to level up.
- **Path Compare** Input two options you're torn between. The system scores each based on growth and risk signals and recommends the better path with reasoning.
- **Next Move** Describe any situation (career, study, finances, stress, relationships) and get a personalised 5-step action plan generated by rule-based logic.
- **Analytics** Visual charts (bar, line, doughnut) showing your habit completions, focus trends, and decision outcomes over the last 7 days.
- **Dashboard** A single-screen summary of all your key metrics and personalized behavioral insights.

## API Reference

All endpoints return JSON. Every authenticated endpoint returns `401` if not logged in.

### Auth
| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/backend/auth/check.php` | Check if session is active |
| `POST` | `/backend/auth/login.php` | `{ email, password }` |
| `POST` | `/backend/auth/register.php` | `{ name, email, password, password_confirm }` |
| `POST` | `/backend/auth/logout.php` | Clear session |

### Habits
| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/backend/api/habits/index.php` | List all habits with streak + weekly % |
| `POST` | `/backend/api/habits/add.php` | `{ name }` |
| `POST` | `/backend/api/habits/update.php` | `{ habit_id }` mark complete today |
| `POST` | `/backend/api/habits/delete.php` | `{ habit_id }` |

### Other Modules
| Endpoint | Notes |
|---|---|
| `/backend/api/focus/index.php` | GET stats, POST `{ duration_minutes }` |
| `/backend/api/decisions/index.php` | GET `?page=N`, POST add, POST `?delete=1` |
| `/backend/api/skills/index.php` | GET list, POST add, POST `?delete=1` |
| `/backend/api/nextmove/index.php` | GET history, POST `{ situation_text }` |
| `/backend/api/pathcompare/index.php` | GET history, POST `{ option_a, option_b }` |
| `/backend/api/dashboard/index.php` | GET aggregated stats + insights |
| `/backend/api/analytics/*.php` | GET chart data (habit_chart, focus_chart, decision_chart) |

## Security

- **Password hashing** `PASSWORD_BCRYPT` via PHP's `password_hash()`
- **SQL injection prevention** PDO prepared statements on every query
- **Session security** `session_regenerate_id(true)` on login, `httponly` cookies
- **Input validation** Both server-side (PHP) and client-side (JavaScript)
- **Directory listing** Disabled via `.htaccess`

## Core JavaScript API

The `FS` global object in `app.js` is the single interface between the frontend and backend:

```javascript
// Make an API call
const res = await FS.api('/api/habits/index.php');
const res = await FS.api('/api/habits/add.php', 'POST', { name: 'Read daily' });

// Check authentication and get user object
const user = await FS.requireAuth(); // redirects to login if not authenticated

// Show a toast notification
FS.toast('Habit added!', 'success');  // types: success | danger | info

// Initialize sidebar + header on any page
FS.initLayout(user, 'habits');  // second arg is the active nav key

// Escape HTML (XSS prevention)
FS.escape(untrustedString);
```
